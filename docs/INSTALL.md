# 🏗️ INSTALL

How to download, test and run.

## Download

You need a client of `git`:
```bash
git clone https://github.com/helvm/helvm-common.git
cd helvm-common
```

## Compile

To compile you need `cabal` and `make`:
```bash
make
```

## By make

```bash
# Start REPL.
make repl

# Generate documentation.
make haddock

# Analyze coverage.
make hpc
```
